# SwachhSankalp

## Purpose
Provides a village roadmap tracking panel for community development, task delegation, and local target progress monitoring.


## Folder Structure
- `components/` - Module-specific UI and village goal check boards.
- `hooks/` - Task status toggle and timeline hooks.
- `services/` - Sync queue interfaces for goal items.
- `types/` - Goal and activity schemas.
- `utils/` - Completion percentages calculations.

## Rules
1. All changes must stay inside `app/swachhsankalp/`.
2. Do not import components or helper code from other feature modules (e.g. `app/gramlipi/`).
3. Only use root-level shared layouts, types, and hooks in `components/`, `types/`, and `hooks/`.
4. Follow strict TypeScript conventions.
